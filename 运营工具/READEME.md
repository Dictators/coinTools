# coinTools使用说明

1. 网络配置

使用coinTools工具前，需要确定网络是否正确，请前往static目录下查看conf.json

````
{
    "ChanConn":  "https://bsc-dataseed1.binance.org",
	"GasLimmit": 210000
}
```

2. 系统支持Excel

当前系统仅支持xlsx格式的excel文件，具体每个功能使用的excel模版请严格参考资源模板文件中的场景excel模版
